# Todo App with Reminders

## Run Instructions

**1.** Extract `env-keys.zip` and copy the contents into a `.env.local` file in the project root. *(Keys will be rotated soon.)*

**2. Install & start:**

```bash
npm i && npm run dev
```

App runs at [http://localhost:3000](http://localhost:3000)

---

> **Note:** The Supabase schema is already applied. Sign up for a new account on the login page to get started.
